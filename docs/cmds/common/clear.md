# clear

Clears your screen if this is possible, including its scrollback buffer cl

## Synopsis

``` bash
clear [-Ttype] [-V] [-x]
```

## Examples

Clear the terminal
``` bash
clear
```

Clear the terminal, do not clear scrollback
``` bash
clear -x 
```
